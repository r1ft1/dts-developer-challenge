# Run the application locally
## Backend 
1. cd backend
2. go mod download
3. go build -o backend
4. ./backend

## Frontend
1. cd frontend
2. touch .env && echo "PUBLIC_API_URL = \\"https://backend-production-93b4.up.railway.app\"" >> .env
3. npm i
4. npm run dev
5. go to http://localhost:5173
